Is This Right for You?

Built for mid-market teams still running WebFocus

This assessment is designed for BI and IT leaders who know a migration is coming — but need clarity before committing to a path.

✅ This is for you if...

  • ✓ Your organisation runs WebFocus (ibi, TIBCO, or OpenText)
  • ✓ You have a licence renewal, cost pressure, or EOL concern on the horizon
  • ✓ You've adopted or are moving to Microsoft Azure or M365
  • ✓ You have 50–2,000 WebFocus reports in production
  • ✓ You need to present a business case internally before committing budget

❌ This is not for you if...

  • ✗ You have no active WebFocus environment
  • ✗ A migration decision has already been made and scoped
  • ✗ You're a vendor or reseller (this is for end-user organisations only)
  • ✗ You need implementation to begin within 30 days
What's Included

Six deliverables. Five business days.

Every assessment is conducted by a senior Prism consultant — not a sales team — and delivered as a written report with a live walkthrough.

01
Environment Inventory
A catalogue of your WebFocus assets — report count, domains, data sources, and Report Caster schedules — so you know exactly what you're migrating.
02
Complexity Classification
Each report category rated Simple / Moderate / Complex based on FOCUS language use, Maintain procedures, Master File dependencies, and chart types.
03
Migration Effort Estimate
A ballpark timeline and effort range (in days) broken into phases: discovery, migration, testing, and user enablement. No padding, no fluff.
04
Risk Register
The top 5–8 risks specific to your environment — data source connectivity, FOCUS logic gaps, licence timing, user adoption — with a mitigation recommendation for each.
05
Target Architecture Sketch
A recommended Azure/Power BI target state diagram — where your data lands, how reports are served, and how your team manages the new environment post-migration.
06
Go / No-Go Recommendation
An honest, written opinion on whether now is the right time to migrate — and if not, what needs to be true before it is. We will tell you to wait if that's what's right.
